# politicalbias-LLMs
LLMs for detection of political bias in Western media sources

Code Folder: Python codes used to 
1. grab data from Guardian (for Guardian articles) and HuggingFace (for BBC articles)
2. perform data preprocessing, standardisation, removal of long articles
3. Within each folder - we have python codes used to perform labelling in batch via prompts using the corresponding LLMs
4. Results visualisation codes

Data Folder: Preped datasets 

Result Folder: Results Visualisation
